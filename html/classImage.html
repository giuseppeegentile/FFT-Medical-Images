Medical Image Processing

Perona-Malik diffusion. Uses the sobel operator implemented (and therefore a 3x3 gauss).

Parameters
dstthe output image
num_iterationsthe number of iteration that will be performed to differentiate the image.
kthe reduction factor, must be in range (0, 1). Higher numbers will reduce more noisy images, but more image information will be lost.
